Buffett Analyst

A Claude Code skill that runs Warren Buffett-style fundamental analysis on any public company — no API key needed, pure yfinance data and a quality/moat/valuation framework.

iWhat it is

Type /buffett-analyst in Claude Code, give it a ticker, and it returns a structured deep-dive: business quality, competitive moat, financial health, intrinsic valuation, and a clear buy/hold/skip verdict.

The skill pulls financials from yfinance (free, no API key) and runs them through Buffett's qualitative framework — does the company have a durable competitive advantage? Does it generate consistent returns on capital? Is the founder still involved? What would Charlie Munger say?

1Quick install

Claude Code
# In Claude Code, after installation:
/buffett-analyst AAPL
/buffett-analyst BRK.B  # meta
/buffett-analyst GOOGL --deep  # extended analysis

Clone github.com/l69d/claude-buffett-analyst and follow the README to wire it into your Claude Code skills directory.

2Framework

Moat assessment

Network effects, brand power, switching costs, cost advantages, intangible assets — does the business have a durable moat?

Quality score

ROE/ROIC history, debt structure, earnings consistency, management quality. Quantitative metrics filtered through qualitative judgment.

Intrinsic value

DCF with conservative assumptions, margin of safety calculation, what price the business would be interesting — not what the market says it's worth.

Obsidian output

Every analysis writes a formatted markdown note to your Obsidian vault for later reference. A searchable library of companies you've studied.
