omnicode

A standalone agentic coding CLI built on Vercel's AI SDK — bring your own model, your own tools, zero lock-in.

Claude GPT Gemini DeepSeek Groq Ollama

iWhat's different from anyclaude?

anyclaude is a proxy — it wraps Claude Code's existing CLI to talk to other models. omnicode is a from-scratch agentic coding CLI built on Vercel AI SDK v5. It has its own REPL, its own tool system, its own file operations — and a pluggable model layer that works with any provider out of the box.

No Anthropic dependency. omnicode doesn't wrap claude. It's a standalone binary you can use with any LLM — including local models via Ollama. The model layer is a single abstraction over all OpenAI-compatible providers.

1Quick start

terminal
# Run with any supported model
npx omnicode
npx omnicode --model gpt-4o
npx omnicode --model gemini-2.5-pro
npx omnicode --model deepseek-v4-flash

# Local models via Ollama
npx omnicode --model ollama/qwen2.5

Set the relevant API key as an environment variable (OPENAI_API_KEY, ANTHROPIC_API_KEY, GEMINI_API_KEY, etc.) and you're off.

2Key features

Pluggable models

Claude, GPT, Gemini, DeepSeek, Groq, Ollama, any OpenAI-compatible endpoint. Switch with a flag.

Own REPL + tools

Read, edit, write, search, bash, web fetch — a full agentic coding environment, not a thin wrapper.

No vendor lock-in

No subscription, no proprietary format. Your config, your keys, your choice of model.

Lightweight

Built on AI SDK v5. No heavy framework, minimal dependencies. Clone and run in seconds.
