sqlgym

101 advanced SQL problems across 14 topics — recursive CTEs, window functions, ranking, pivoting, nulls, and beyond. In-browser SQLite with gated worked solutions.

iWhat it is

sqlgym is LeetCode-style SQL practice that runs entirely in your browser — no setup, no database credentials, no backend. Powered by sql.js + WASM, every query executes against a real SQLite instance in the browser tab.

Each problem has a schema preview, a prompt, and an inline editor. Submit your query and see results instantly. Stuck? Each problem has 1-3 worked approaches with step-by-step explanations.
